# ICE-FIRE Particle Hand Tracking Effect

An interactive, real-time computer vision application that assigns "Fire" and "Ice" particle effects to your hands using your webcam.

## Features
- **Fire & Ice Elements**: The left hand gets a fire effect, and the right hand gets an ice effect.
- **Particle System**: Realistic, dynamic particles that orbit and follow your hands.
- **Writing Mode**: Point your index finger to enter drawing mode and sketch glowing trails in the air.

## How to Use

### 1. Install Dependencies
Make sure you have python installed, then run the following command to install the required libraries:
```bash
pip install -r requirements.txt
```

### 2. Run the Script
Execute the script using python:
```bash
python main.py
```

### 3. Controls & Gestures
- **Show hands**: Bring your left or right hand into the camera view.
- **Write**: Point your index finger to draw glowing lines.
- **Q**: Quit the application.
- **R**: Reset the particles and canvas.
